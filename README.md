# Go-Free-Projet

Présentation générale

Go-Free-Projet est une application web développée dans le cadre de la préparation et de la validation du titre RNCP.
Ce projet s’inscrit dans une démarche de mise en pratique des compétences acquises en développement web full-stack, notamment en conception, développement et structuration d’une application complète.

Objectifs du projet

L’objectif principal de ce projet est de :

Concevoir et développer une application web fonctionnelle
Mettre en œuvre une architecture client-serveur
Appliquer les bonnes pratiques de développement (organisation du code, séparation des responsabilités, structuration front-end / back-end)
Démontrer la maîtrise des technologies web modernes

Architecture technique

Le projet est structuré en deux parties distinctes :

1. Partie Front-end

La partie front-end constitue l’interface utilisateur de l’application.
Elle est développée à l’aide des technologies suivantes :

HTML : structuration du contenu
CSS : mise en forme et design de l’interface
JavaScript : gestion des interactions et communication avec l’API
Cette partie permet à l’utilisateur d’interagir avec l’application de manière intuitive et dynamique.

2. Partie Back-end

La partie back-end assure la gestion de la logique métier et le traitement des données.
Elle est développée avec :

Node.js : environnement d’exécution JavaScript côté serveur
Express.js : framework permettant la création d’API et la gestion des routes

Le back-end prend en charge :

La gestion des requêtes HTTP
La communication avec la base de données
La sécurisation et la structuration des échanges de données
